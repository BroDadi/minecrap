# minecrap
a very crappy clone of minecraft, built upon [coding with sphere's opentk minecraft tutorials](https://www.youtube.com/playlist?list=PLvN4CrYN-8i6fPOfPgS8YsoncOYZql9p8). currently sucks a lot

includes:
* a PLAYER with COLLISIONS
* a CAMERA
* a GUI
* a 16x16 chunk world with terrain generation (it's just basic noise that i didn't even edit it's just from the tutorial)
* a whopping FOUR (4) types of blocks (or FIVE (5) if you include air!)
